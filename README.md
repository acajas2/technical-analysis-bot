# Project Overview (Technical-Analysis-Bot)
This project is a Python-based trading tool that will analyze Market data and print signals using:
- RSI
- Stochastic RSI
- SMA Indicators
The bot retries market data using:
- yfinance
- Bloomberg API (work in progress)
Current functionalities include:
- coded RSI, StochRSI, and SMA indicators using functions to print results
- returns results for inputed ticker symbol (can be any ticker)
- provides interpretation of indicators for inputted stock
Future plans:
- Backtester to test when to buy/sell stock
- Chart implementation
- Dashboard visualization (multiple charts, indicators)
- Price targets, and notification pings
